# Econometrics
A desktop tool for recording a company’s financial metrics, 
followed by data processing and analysis through graphical visualization 
and the use of regression methods such as linear, polynomial, and random forest.

### Features
- Database that contains financial metrics.
- Treeview visualization of financial metrics.
- Graphical data visualization method (boxplot).
- Regression analysis.

### Technologies
- Python  
- CustomTkinter / Tkinter — GUI design  
- SQLite3 — database management  
- Pandas, NumPy — data handling  
- Matplotlib — visualization  
- Scikit-learn — regression algorithms  
- PIL — interface images and icons
